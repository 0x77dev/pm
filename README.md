# Usage
<!-- usage -->
```sh-session
$ npm install -g pm
$ pm COMMAND
running command...
$ pm (-v|--version|version)
pm/0.0.1 darwin-x64 node-v12.0.0
$ pm --help [COMMAND]
USAGE
  $ pm COMMAND
...
```
